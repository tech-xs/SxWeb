## 项目启动流程

> **1.安装库**\
> **2.配置接口请求根路径**\
> **3.运行项目**

#### 1.安装库

```shell
npm i yarn -g
yarn install
```

### 2.配置接口请求根路径

> 修改项目根目录下的[.env.dev](.env.dev) [.env.prod](.env.prod)文件中的[VUE_APP_BASE_URL]参数\
> 需要修改的文件 [.env.dev](.env.dev) [.env.prod](.env.prod)\
> 需要修改的参数 VUE_APP_BASE_URL

### 3.运行项目

```
yarn run dev
```
